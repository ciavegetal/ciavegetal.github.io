## 🚀 Hosting on GitHub Pages

### Step 1 — Upload to GitHub
1. Go to [github.com](https://github.com) and sign in (or create a free account)
2. Click **"New repository"**
3. Name it `ciavegetal-website` (or any name you prefer)
4. Set visibility to **Public** (required for free GitHub Pages)
5. Click **"Create repository"**
6. Upload all files from this folder using **"uploading an existing file"**

### Step 2 — Enable GitHub Pages
1. In your repository, go to **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **"Deploy from a branch"**
3. Select branch: `main` | Folder: `/ (root)`
4. Click **Save**
5. Wait ~60 seconds, then your site is live at:  
   `https://YOUR-USERNAME.github.io/ciavegetal-website/`

### Step 3 — Custom Domain (optional, recommended)
To use `ciavegetal.com.br` or `www.ciavegetal.com.br`:
1. Edit the `CNAME` file — replace the placeholder with your actual domain
2. In your domain registrar (GoDaddy, Registro.br, etc.), add a DNS record:
   - **Type:** CNAME  
   - **Name:** `www`  
   - **Value:** `YOUR-USERNAME.github.io`
3. In GitHub Pages settings, enter your custom domain and enable **"Enforce HTTPS"**
